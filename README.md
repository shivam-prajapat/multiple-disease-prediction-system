<div align="center">

### 💀 Disease Detection Gap

</div>

🔵 **Problem Statement**

Millions of people worldwide suffer from undiagnosed chronic diseases due to limited access to early screening tools, resulting in catastrophic health and economic consequences:

| 🔴 $300B ANNUAL IMPACT | 🟠 70% LATE DIAGNOSIS | 🟡 2M+ PREVENTABLE DEATHS | ⬛ LIMITED EARLY SCREENING |
|---|---|---|---|
| **Economic Burden** | **Missed Early Stage** | **Annual Mortality Risk** | **Poor Accessibility** |
| *Healthcare costs from undiagnosed diabetes, heart disease, and neurological disorders* | *Patients diagnosed at advanced stages when treatment is harder* | *Deaths preventable with early intervention* | *Lack of affordable AI-powered screening tools* |

The healthcare system desperately needs:

- 🔬 Real-time AI-powered disease risk screening accessible to everyone
- 💡 Predictive models that consider multiple clinical biomarkers simultaneously
- 🧬 Multi-disease coverage — Diabetes, Heart Disease, and Parkinson's in one platform
- 🚨 Instant risk alerts that guide patients to seek timely medical attention
- 📊 Transparent model accuracy so users understand the tool's limitations

## 🚀 Getting Started

### Prerequisites

```bash
Python 3.9+
pip (latest version recommended)
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/multiple-disease-prediction-system.git
cd multiple-disease-prediction-system

# 2. Install dependencies
pip install -r requirements.txt

# 3. (Optional) Retrain models with latest scikit-learn
python retrain_models.py

# 4. Launch the app
streamlit run "multiple disease pred.py"
```

### 🌐 The app will open at `http://localhost:8501`

## 🧠 Model Architecture

```
MediPredict AI
│
├── 🩸 Diabetes Model
│   ├── Algorithm: Support Vector Machine (RBF Kernel)
│   ├── Dataset: Pima Indians Diabetes (768 samples)
│   ├── Features: 8 clinical parameters
│   └── Test Accuracy: ~77%
│
├── ❤️ Heart Disease Model
│   ├── Algorithm: Logistic Regression
│   ├── Dataset: Cleveland Heart Disease (303 samples)
│   ├── Features: 13 cardiovascular parameters
│   └── Test Accuracy: ~79%
│
└── 🧠 Parkinson's Model
    ├── Algorithm: Support Vector Machine (RBF Kernel)
    ├── Dataset: UCI Parkinson's Dataset (195 samples)
    ├── Features: 22 voice biomarkers
    └── Test Accuracy: ~87%
```
